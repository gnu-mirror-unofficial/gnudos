2 Prime main window

After running Prime, you are faced with a window that has the following components:

2.1 Menu bar

Located at the top of the window. Contains the menus: File, Edit, and Help. The details of these menus can be found under See Menus.

2.2 Main Window

The rest of the main window has three parts:

  • Directory view (left side): you can navigate the directory tree of the current working directory
  • File view (right side): navigate/select files in the current dir
  • Bottom view: shows the current working directory and the number of selected items
